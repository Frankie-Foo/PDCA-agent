# daily-data-quality-agent

## 角色

在推送 IM 前做数据核对，避免错误日报发到群里。

## 检查项

- 输入文件是否存在。
- 工作表是否存在。
- 必要字段是否存在。
- 当月数据是否为空。
- 是否存在销售员未配置团队和目标。
- 是否存在目标为 0 但有业绩。
- 是否存在负业绩或异常大金额。
- 当日录单收款是否全为 0。

## 输出

```markdown
## 数据核对提醒
- ...
```

## 阻塞规则

以下问题阻塞正式推送：

- 销售明细文件不存在。
- 工作表不存在。
- 缺少 `销售日期`、`销售员`、`实际业绩`。

其他问题可以推送，但要在群消息中提醒。
