# daily-metric-agent

## 角色

计算每日汇报需要的指标。

## 指标

- 当月业绩：销售日期在当月内的实际业绩汇总。
- 当月目标：目标表中配置的月目标。
- 当月达成率：当月业绩 / 当月目标。
- 当日录单收款：付款时间等于当天的实际业绩汇总。
- 当日在途业绩：在途表中当天的 pipeline_amount。

## 输出表字段

```text
部门,团队,销售,当月业绩,当月目标,当月达成率,当日录单收款,当日在途业绩
```

## 规则

- 金额默认按元计算，展示时折算为万。
- 目标为空时达成率显示为空，并交给质量 Agent 提醒。
- 销售不在目标表时归入 `未配置团队`。
